# 🛒 Online Retail Sales Dashboard

## 📌 Project Overview
This project analyzes the Online Retail Sales dataset (2010–2011) using Power BI. The dataset was cleaned using Python (Jupyter Notebook) and transformed into an interactive dashboard to identify sales trends, customer behavior, and country-wise revenue.

---

## 🛠️ Tools Used
- Python
- Jupyter Notebook
- Pandas
- Power BI
- Microsoft Excel

---

## 📂 Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Data Visualization
5. Dashboard Development
6. Business Insights

---

## 📊 Dashboard Features
- Total Revenue KPI
- Total Quantity KPI
- Total Customers KPI
- Monthly Revenue Trend
- Revenue by Country
- Revenue by Product Description
- Country Filter (Slicer)
